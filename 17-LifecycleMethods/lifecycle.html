React Lifecycle Methods

Built-in React methods for mounting, updating and unmounting and some less common methods.

Lifecycle

Every component comes with methods that allows devs to update application state and reflect the changes to the UI before/after key react events.

The three main phases that a component goes through are mounting, updating and unmounting.

When a component is told to be put on the page, React first runs the constructor, then the render(), which tells React what should be displayed. React updates the DOM to math the output of render(). Then componentDidMount() runs after,

componentDidMount() is a a good place to load any data via AJAX or set up subscriptions/timers.
