Color Project Notes

New Technologies

  • Material UI - very popular library for React components that combine CSS and JavaScript.
  • Chroma.js - library for interacting with colors, including the rgba and hexa colors. Used also for chaning the color saturation.
  • Emoji-Mart - library for picking emojis
  • React Sortable HOC - simple way of implementing drag and drop in our color palatte.
  • React Copy-to-Clipboard - does what the title suggests.
  • React PureComponent - built-in to react that prevents unnecessisary component re-renders
  • React-Form-Validator-Core - validators to help us ensure that the users color palette names are valid and not duplicates
  • React-transition-group - will help us transition between our various routes.
  • JSS - Write JavaScript to describe our styles. Will be in .js files.
